## How to use

### Require the module

```javascript
var onehostjs = require("@1hostjs/hosting");
```

### Use it

```javascript
onehostjs(modulesJson, dirname);
// modulesJson should be the modules.json file for 1hostjs
// dirname should be __dirname
```
