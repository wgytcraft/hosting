You can install 1host.js with Yarn or npm.

<d-code-group>
  <d-code-block label="Yarn" active>
    
  ```bash
  yarn add @1hostjs/hosting
  ```
  </d-code-block>
  <d-code-block label="npm">
  
  ```bash
  npm install @1hostjs/hosting
  ```
  </d-code-block>
</d-code-group>
